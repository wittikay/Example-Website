# Example Website

A responsive, modern landing page demo featuring layered sections, a pop-out navigation menu, and an animated ASCII renderer.

## Features

- Responsive layout with three main sections and a footer
- Pop-out header navigation
- Modern UI accents (chips, CTA buttons, stats)
- Animated ASCII renderer with click-drag rotation

## Structure

- `index.html` – page structure and JavaScript
- `styles.css` – styling and layout

## Usage

Open `index.html` in a browser.

## Notes

If you want to swap the ASCII model, replace the renderer logic in `index.html`.
