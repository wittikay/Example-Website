# Example Website

A responsive, modern landing page demo featuring layered sections, a pop-out navigation menu, and an animated ASCII renderer.

## Features

- Responsive layout with three main sections and a footer
- Pop-out header navigation
- Modern UI accents (chips, CTA buttons, stats)
- Animated ASCII renderer with click-drag rotation

## Structure

- `index.html` – page structure
- `css/styles.css` – styling and layout
- `js/main.js` – app entry point
- `js/nav.js` – navigation interactions
- `js/ascii.js` – ASCII renderer logic

## Usage

Find the live page here: https://wittikay.github.io/Example-Website/

## Notes

If you want to swap the ASCII model or behavior, update `js/ascii.js`.
